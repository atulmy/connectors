	USAGE:
	
	Include Jquery file
	<script src="jquery.min.js"></script>
	
	Include Connector Plugin file
	<script src="jquery.connector.js"></script>
	
	Onload Script
	<script>
		$( function() {
			$('#container').connector({
				connectFrom: '#box1', 
				connectTo: '#box2'
			});
		});
	</script>
	
	where
	connectFrom = ID or Class of Box FROM which you need to connect the path
	connectTo = ID or Class of Box TO which you need to connect the path
	'#container' = All paths will be drawn inside this element
	
	Optional parameter
	dotSize: 5
	
	The path obtained is not a curve one and hence may not be practically used. 
	Also since there is lot of looping for tracing path, we generate lot of <div> element which in turn makes DOM heavy. 
	Instead of this, for modern browsers, we can use HTML5 Canvas version of Connector (check top menu for link).
